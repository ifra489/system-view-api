# System View API

A simple Node.js API that provides system information.

## Features

- CPU Information
- Memory Usage
- Battery Status
- Network Information
- OS Information
- Process Information




## Routes

- `/cpu` – Get CPU details  
- `/memory` – Get memory usage  
- `/battery` – Get battery status  
- `/network` – Get network information  
- `/process` – Get running processes  
- `/os` – Get OS details  

## Run Project

1. Install dependencies:

```bash
npm install
```
2. Start the server:
```
node api.js
```
```
Server runs at: http://localhost:5000
```
